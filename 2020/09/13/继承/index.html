继承

  • 继承描述了Java类与类之间的关系，在Java中一个类只能继承另一个类，不过可以实现多级继承
  • 格式： public class 子类名 extends 父类名{ }
  • 下面我们来看两个问题：

继承中，子类继承了父类的哪些属性和方法？
这个答案是： 所有
为什么是所有呢？ 因为这里是一个权限问题，我们只能是说不够权限访问不能说是没有继承，你可以拥有，不一定够权限使用。
